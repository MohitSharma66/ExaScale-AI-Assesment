## 🚀 Assignments

### Assignment 1 – Intelligent Power Demand Forecasting

A machine learning solution designed to forecast power demand using historical consumption data and predictive analytics techniques.

**Includes:**

* Data preprocessing pipeline
* Forecasting model implementation
* API integration
* Dockerized deployment
* Documentation and usage guide

---

### Assignment 2 – GHG Emissions Reporting Platform

A platform for collecting, processing, and reporting greenhouse gas (GHG) emissions data with a focus on scalability and maintainability.

**Includes:**

* Backend architecture
* REST API endpoints
* Emissions data processing workflows
* Docker support
* Detailed technical documentation
